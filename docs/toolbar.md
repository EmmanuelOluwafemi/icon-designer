# Editor Layout

The Icon Builder editor is structured into five main UI regions.

1. Top Bar
2. Left Toolbar
3. Left Sidebar
4. Canvas
5. Right Properties Panel

## Design Principles

- Familiar to designers coming from Figma
- Optimized for icon pack workflows
- Minimal tool clutter
- Focus on frames and icon variants
